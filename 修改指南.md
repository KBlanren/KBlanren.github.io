> 先记住：**内容**改 Markdown 文件，**导航/样式**改 `index.html` 和 `css/style.css`，**交互逻辑**改 `js/main.js`。三者分工明确。

## 二、逐项详解

### 2.1 添加一篇新笔记（最常用）

**两步**：写内容 → 加导航。

**第一步：写内容**

在 `content/` 下找到对应文件夹，新建 `.md` 文件。例如新建一篇「行为金融学」笔记到公开课：

```
content/courses/
├── advanced-macro.md    （已有）
├── econometrics.md      （已有）
├── finance-theory.md    （已有）
└── behavioral-finance.md   ← 新建这个
```

用 Markdown 写内容，LaTeX 公式直接写：

```markdown
# 行为金融学

前景理论的价值函数：

$$ v(x) = \begin{cases} x^\alpha & x \geq 0 \\ -\lambda (-x)^\beta & x < 0 \end{cases} $$

其中 $\lambda > 1$ 表示损失厌恶。
```

**第二步：在导航栏加链接**

打开 `index.html`，找到「第壹辑 · 公开课笔记」对应的手风琴 `panel-1`：

```html
<!-- 第壹辑 -->
<li class="accordion-wrap">
    <button class="accordion-trigger" data-panel="panel-1">
        <span class="section-ribbon">第壹辑 · 公开课笔记</span>
        <span class="arrow">▸</span>
    </button>
    <ul class="accordion-panel" id="panel-1">
        <li class="index-item"> ... 高级宏观 ... </li>
        <li class="index-item"> ... 计量 ... </li>
        <li class="index-item"> ... 金融 ... </li>
        <!-- 在这里新增一行： -->
        <li class="index-item">
            <a href="#courses/behavioral-finance" class="nav-link" data-path="courses/behavioral-finance">
                <span class="tab-mark">▪</span>
                <span class="tab-text">行为金融学</span>
                <span class="tab-year">25</span>
            </a>
        </li>
    </ul>
</li>
```

**关键规则**：
- `href="#courses/behavioral-finance"` → 决定浏览器地址栏的 `#` 后缀
- `data-path="courses/behavioral-finance"` → 决定 JS 去加载 `content/courses/behavioral-finance.md`
- 这两个值必须完全一致，否则导航高亮和内容加载会错位

保存，刷新页面即可看到新笔记。

### 2.2 新建一个全新板块（第伍辑）

**三步**：新建文件夹 → 改导航 → 改 JS 映射。

**第一步**：新建内容文件夹

```bash
# 在 content/ 下新建
mkdir content/behavioral

# 写第一篇笔记
echo "# 行为金融学导论\n\n内容..." > content/behavioral/intro.md
```

**第二步**：在 `index.html` 里新增一个手风琴块

在最后一个 `</li>`（第肆辑的结尾）之后，加一个新的 `accordion-wrap`：

```html
<!-- 第伍辑 -->
<li class="accordion-wrap">
    <button class="accordion-trigger" data-panel="panel-5">
        <span class="section-ribbon">第伍辑 · 行为金融</span>
        <span class="arrow">▸</span>
    </button>
    <ul class="accordion-panel" id="panel-5">
        <li class="index-item">
            <a href="#behavioral/intro" class="nav-link" data-path="behavioral/intro">
                <span class="tab-mark">▪</span>
                <span class="tab-text">行为金融学导论</span>
                <span class="tab-year">25</span>
            </a>
        </li>
    </ul>
</li>
```

**注意**：`data-panel="panel-5"` 和 `id="panel-5"` 必须是唯一的，不能和已有的 `panel-1` 到 `panel-4` 重复。

**第三步**：在 `js/main.js` 中更新映射

找到 `openAccordionForPath` 函数（约第 88 行），在 `panelMap` 里新增一行：

```javascript
const panelMap = {
    'courses': 'panel-1',
    'papers': 'panel-2',
    'blog': 'panel-3',
    'dsge': 'panel-4',
    'behavioral': 'panel-5'   // ← 新增这行
};
```

同时在 `PATH_META` 中也要加一条，让页面头部显示正确的分类：

```javascript
const PATH_META = {
    'home': { category: '总览', date: '2025' },
    // ... 已有的 ...
    'behavioral/intro': { category: '行为金融', date: '2025' }   // ← 新增
};
```

### 2.3 改颜色 / 纸张质感

打开 `css/style.css`，前 15 行就是全部颜色变量：

```css
:root {
    --desk: #2a1f15;          /* 桌面背景色 */
    --desk-light: #3a2a1a;    /* 桌面浅色渐变 */
    --paper: #f5e6c8;          /* 主纸张颜色 */
    --paper-dark: #ebd9b0;     /* 纸张深色（卡片、代码块） */
    --paper-edge: #e0d0a8;     /* 边框、分隔线 */
    --ink: #2c1810;            /* 主文字颜色 */
    --ink-light: #5a3d2b;      /* 次要文字 */
    --ink-faded: #8b7355;      /* 淡化文字（年份、脚注） */
    --accent: #8b4513;         /* 强调色（棕色） */
    --accent-red: #a04030;     /* 印章红 */
    --gold: #b8860b;           /* 金色装饰 */
    --stamp-red: #a03020;      /* 审阅印章红 */
}
```

直接改数值，全站颜色会自动联动。例如把纸张改得更白：

```css
--paper: #faf6ed;
```

或者把桌面改成更偏黑的棕色：

```css
--desk: #1e1510;
```

### 2.4 改字体

打开 `css/style.css` 第 3 行：

```css
@import url('https://fonts.googleapis.com/css2?family=Noto+Serif+SC:wght@400;600;700&family=Playfair+Display:ital,wght@0,400;0,700;1,400&display=swap');
```

这是 Google Fonts 的导入。你可以去 [fonts.google.com](https://fonts.google.com) 选其他字体，替换这行链接。然后在文件里搜索 `font-family` 按需替换。

如果你想用本地字体（不用翻墙加载），把 `@import` 删掉，改用 `font-family` 直接写系统字体：

```css
body {
    font-family: 'Songti SC', 'SimSun', 'STSong', 'serif';
}
```

### 2.7 改 LaTeX 配置

打开 `index.html` 第 8–17 行：

```html
<script>
    window.MathJax = {
        tex: {
            inlineMath: [['$', '$'], ['\(', '\)']],
            displayMath: [['$$', '$$'], ['\[', '\]']],
            processEscapes: true,
            processEnvironments: true
        },
        options: { skipHtmlTags: ['script', 'noscript', 'style', 'textarea', 'pre'] }
    };
</script>
```

- `inlineMath` 定义行内公式的分隔符。默认 `$...$` 和 `\(...\)` 都支持。
- `displayMath` 定义独立公式的分隔符。默认 `$$...$$` 和 `\[...\]` 都支持。

如果你习惯用其他写法（比如 `\begin{equation}`），可以在这里加。但 `$...$` 和 `$$...$$` 是写 Markdown 最顺手的，不建议改。

## 四、常见错误

| 现象 | 原因 | 修复 |
|------|------|------|
| 点击导航没反应 | `href` 和 `data-path` 不一致 | 检查两者是否相同 |
| 页面显示「卷宗未找到」 | `content/xxx/xxx.md` 文件不存在 | 新建对应的 Markdown 文件 |
| 公式不渲染 | MathJax CDN 没加载 | 检查网络，或换 CDN 源 |
| 左侧导航没展开 | 路径前缀没加入 `panelMap` | 在 `js/main.js` 的 `panelMap` 中加映射 |
| 导航栏太宽/太窄 | 改了 `filing-index` 宽度但内容溢出 | 缩短 `tab-text` 的文字，或减小字号 |

---

> 一句话总结：内容 = Markdown 文件，骨架 = index.html，皮肤 = style.css，脑子 = main.js。三部分独立，互不影响。
