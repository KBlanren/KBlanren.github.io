# 首页简介 + 贡献页（Contributors）—— 逐行修改教程

> 目标：
> 1. 改首页简介为「这是我和我朋友们的唐网站……」
> 2. 导航栏新增 CONTRIBUTORS 链接
> 3. 新增贡献页（团队页），展示 蓝人王 和 FIN 两个人
> 4. 照片放在 `images/` 文件夹，按人名命名，网页自动读取
> 5. 以后想增加人，只需要改 `js/main.js` 里的 `CONTRIBUTORS` 数组 + 放照片

### 2.4 新增 `renderContributors()` 函数（在 `loadHome()` 之后）

```js
function renderContributors() {
    updateHeader('contributors');   // 更新顶部标签为 "CONTRIBUTORS :: 2025"

    // 遍历 CONTRIBUTORS 数组，生成每个人的卡片 HTML
    const membersHtml = CONTRIBUTORS.map(person => `
        <div class="contributor-card">
            <div class="contributor-avatar-wrap">
                <img src="${person.photo}" alt="${person.name}" class="contributor-avatar" onerror="this.style.display='none';this.parentElement.innerHTML+='<div class=\\'contributor-avatar-placeholder\\'>?</div>'">
            </div>
            <div class="contributor-role">${person.role}</div>
            <div class="contributor-name">▶ ${person.name}</div>
        </div>
    `).join('');

    // 组装整个页面：大标题 + 人员网格
    contentArea.innerHTML = `
        <div class="contributors-page">
            <h1 class="contributors-title">Our Team</h1>
            <div class="contributors-grid">${membersHtml}</div>
        </div>
    `;

    // 触发 MathJax 渲染（万一 role 里有公式）
    if (window.MathJax) {
        MathJax.typesetPromise([contentArea]).catch(() => {});
    }
}
```

> 逐行解释：
> - `CONTRIBUTORS.map(...)`：把数组里的每个人转换成一段 HTML，`.join('')` 把数组拼接成一个大字符串
> - `contributor-card`：每个人的外框，包含头像 + 角色 + 名字
> - `onerror="..."`：如果照片加载失败（文件不存在或路径错误），隐藏图片，显示一个 `?` 占位符。这样你还没放照片时页面不会崩，而是显示问号
> - `▶`：就是图片里那个箭头符号，像素风替代方案
> - `contributors-grid`：用 flex 布局，让两个人并排居中

## 四、css/style.css 改了什么

在「分类列表样式」和「MathJax」之间，新增了一大段贡献页样式（约第 870–930 行）：

```css
/* ===== 贡献者页面（像素风团队页） ===== */
.contributors-page { ... }          /* 整体居中 */
.contributors-title { ... }         /* "Our Team" 大标题，像素字体 */
.contributors-grid { ... }          /* 人员网格，flex 横向居中排列 */
.contributor-card { ... }           /* 单人卡片，纵向 flex */
.contributor-avatar-wrap { ... }    /* 头像外框，180x180，4px 边框，圆角 */
.contributor-avatar { ... }         /* 头像图片本身，object-fit: cover */
.contributor-avatar-placeholder { ... }  /* 照片加载失败时显示的 ? 占位符 */
.contributor-role { ... }           /* 职位文字 */
.contributor-name { ... }           /* 名字 + ▶ 箭头 */
```

### 关键样式逐行解释

| 选择器 | 作用 |
|--------|------|
| `.contributors-grid { display: flex; gap: 48px; }` | 两个人横向并排，中间间隔 48px，自动居中 |
| `.contributor-avatar-wrap { border-radius: 16px; }` | 头像圆角，参考图片的圆角矩形风格 |
| `.contributor-avatar { image-rendering: pixelated; }` | 照片用像素化渲染，保持网站整体像素风 |
| `onerror`（在 JS 里） | 如果 `images/蓝人王.jpg` 不存在，显示 `?` 占位符，不会留空白 |

### 移动端适配（在 @media 里新增）

```css
@media (max-width: 768px) {
    ...
    .contributor-card { width: 160px; }       /* 手机上网格变窄 */
    .contributor-avatar-wrap { width: 140px; height: 140px; }  /* 头像缩小 */
    .contributor-name { font-size: 0.6rem; }  /* 名字字号缩小 */
}
```

## 五、照片怎么放

### 步骤

1. 把你的照片文件放进 `images/` 文件夹
2. 文件名必须和 `CONTRIBUTORS` 数组里的 `photo` 字段一致：
   - `images/蓝人王.jpg`
   - `images/FIN.jpg`

3. 刷新网页，CONTRIBUTORS 页面上就会显示照片

### 如果照片没显示

- 检查文件名是否完全一致（包括大小写、扩展名）
- 检查照片格式：`.jpg`、`.png`、`.jpeg` 都可以，但 `photo` 路径要写对扩展名
- 打开浏览器控制台（F12 → Console），看有没有红色的 404 报错
- 如果看到 `?` 占位符，说明文件没找到，检查路径和文件名

### 改照片格式或扩展名

如果 FIN 的照片是 `.png` 而不是 `.jpg`，改 `js/main.js` 里的 `photo` 路径：
```js
{
    name: 'FIN',
    role: 'Co-Founder & Horror Writer',
    photo: 'images/FIN.png'   // 改这里
}
```

## 六、以后想增加人怎么做

### 三步：

1. **放照片**：把新照片放进 `images/` 文件夹，比如 `images/张三.jpg`
2. **改数组**：在 `js/main.js` 的 `CONTRIBUTORS` 数组里加一段：
   ```js
   {
       name: '张三',
       role: 'Contributor',
       photo: 'images/张三.jpg'
   }
   ```
3. **刷新网页**：CONTRIBUTORS 页面上自动出现第三个人

> 不需要运行 `build.py`，不需要改 `manifest.json`，不需要改 HTML 或 CSS。

### 想改角色名

直接改 `role` 字段：
```js
role: 'Managing Director & Editor-in-Chief'
```

### 想改箭头符号

在 `renderContributors()` 函数里找到这一行：
```js
<div class="contributor-name">▶ ${person.name}</div>
```
把 `▶` 改成你想要的符号，比如 `→` 或 `◆`。

## 八、常见问题

**Q：CONTRIBUTORS 页面显示两个问号？**
> 说明你还没放照片。`images/` 文件夹下需要有 `蓝人王.jpg` 和 `FIN.jpg`。放完照片刷新即可。

**Q：照片显示了但是变形了？**
> 照片比例和头像框（180x180 正方形）不一致。建议把照片裁剪成接近正方形的比例，或者 CSS 里改 `object-fit: contain`（但可能留白）。

**Q：我想让照片是圆形而不是圆角矩形？**
> 改 `css/style.css`：
> ```css
> .contributor-avatar-wrap {
>     border-radius: 50%;   /* 把 16px 改成 50% */
> }
> ```

**Q：三个人怎么排？一排三个还是一排两个？**
> 默认 `flex-wrap: wrap`，会自动换行。如果屏幕够宽，三个会并排；如果屏幕窄，会换行。不需要改代码。
